# SQLiteAdapter

A simple wrapper around the SQLite3 API. Written in Swift 5.

Usage examples can be found in the [iOS-MVVM-Clean-Architecture](https://github.com/denissimon/iOS-MVVM-Clean-Architecture) demo app.
